Theme Selection and Scenario

I selected the Astra theme for my WordPress website because it is a lightweight, highly customizable, and beginner-friendly theme. Astra offers excellent speed performance and seamless integration with page builders like Elementor, which makes designing easy without needing to code.

For my scenario — a personal portfolio website — Astra is perfect because it provides clean, modern layouts that highlight projects, skills, and contact information professionally. Its responsive design and SEO compatibility make it ideal for showcasing a professional identity online.
